Install

Dependencies

Angular-ts uses yeoman so to install angular-ts you need Nodejs and NPM installed.

After you installed Node and NPM you need to install some packages needed by angular-ts to create and build the project.

  • Yeoman
  • Gulp
  • Bower
  • Typings


After you installed the dependencies yo can install the angular-ts generator.




Now you have everything you need to create your angular-ts project.
