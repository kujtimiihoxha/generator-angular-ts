Angular Typescript project generator

Make Angular 1 feel like Angular 2

Angular-ts is a project starter that mimics the angular 2 syntax but lets you use the existing angular 1 plugins. It is already fully configured and uses gulp as a build manager. Angular-ts has its own generator that will create the project and make it easy to add additional elements to your angular application. Although this starter is very easy to understand and it is aimed to all categories of developers (starter-advanced) it is recommended that you gain some basic knowledge of Angular and Typescript before using this starter.

If you want to contribute to angular-ts please fork angular-ts or the generator-angular-ts repositories and submit pull requests.

  • Angular Typescript Documentation/Example project: angular-ts
  • Angular Typescript Generator: generator-angular-ts
