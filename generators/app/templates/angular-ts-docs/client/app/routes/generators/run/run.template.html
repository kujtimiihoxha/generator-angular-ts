Run Generator

The run generator is used to create angular run functions.

To create a run function execute the following.



Options


    
    
  • name :  file name  [REQUIRED]
    • 
    
  • inject : dependencies to inject in the config
    • 



Create a run


To create a run ex. foo run execute




After the generator finishes one file will be created.


    
    
  • foo.run.ts
    • 
    
    



The run file will be created in the run folder.




Inject


The inject option will tell the generator to inject dependencies.




Now $state will be injected in the run.
