Filter Generator

The filter generator is used to create angular filters.

To create a directive execute the following.



After the generator finishes one file will be created.


Options


    
    
  • param : adds additional parameters to the filter function
    • 



Create a filter


To create a filter ex. baz filter run




    
    
  • baz.filter.ts: filter function
    • 
     
    



The filter will be created in the filters folder.




Besides the filter the generator automatically adds a test to the test folder.




    
    
  • baz.filter.spec.ts
    • 
    
    





    
Note

    
If you don't want to create the test with the filter just add --noTest flag after the options of the generator.




Param


The param option will tell the generator to add additional parameters to the filter function.
