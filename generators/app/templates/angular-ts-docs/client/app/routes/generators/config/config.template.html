Config Generator

The config generator is used to create angular configuration functions.

To create a config function execute the following.



Options


    
    
  • name :  file name  [REQUIRED]
    • 
    
  • inject : dependencies to inject in the config
    • 



Create a config


To create a config ex. foo config run




After the generator finishes one file will be created.


    
    
  • foo.config.ts
    • 
    
    



The config file will be created in the config folder.




Inject


The inject option will tell the generator to inject dependencies.




Now $urlRouterProvider will be injected in the config.
